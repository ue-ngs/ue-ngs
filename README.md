# UE長崎

島を繋ぎ、長崎を一つのクリエイティブ・ギルドへ。

UE長崎は、Unreal Engine を中心としたリアルタイム3D技術を通じて、長崎のクリエイティブな個人たちを繋ぐコミュニティです。

## ビジョン

### 「離島」を「個」の象徴に変える

長崎の島々は物理的に離れていますが、かつてのあなたのように「ITに触れたい」と願う個人もまた、周囲から理解されず「情報の孤立（島）」にいるはずです。

### 「分散」こそが最強のネットワーク

「一箇所に集まらなければならない」という制約を捨てます。UEやメタバースの技術を使えば、対馬にいても、五島にいても、長崎市内にいても、同じプロジェクトで「世界」を作れます。

### 「書き出す（アウトプット）」を継続のガソリンにする

パートナーの方の「環境をつくる」「タネを拾う」という思いを、「ギルド（職人組合）」という言葉に込めました。
